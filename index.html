CP Algoritmos

O objetivo deste projeto é traduzir e tornar acessível os recursos http://e-maxx.ru/algo e https://cp-algorithms.com, para ajudar estudantes fornecendo descrições de algoritmos e estrutura de dados populares no campo de programação competitiva(en.wikipedia.org/wiki/Competitive_programming).

    • Equação Diofantina Linear

Grafos

  • Travessia em grafos

  • Caminhos curtos de todos os pares
